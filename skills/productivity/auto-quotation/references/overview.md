# 报价单自动生成系统 — 完整方案

## 整体架构

```
你发微信："给XX客户报10台设备A的价"
       │
       ▼
   ┌─────────────────────┐
   │  小马（Hermes AI）   │
   │  • 理解需求         │
   │  • 查询SQLite数据库  │
   │  • 生成报价单       │
   └────────┬────────────┘
            │
    ┌───────┴───────┐
    │               │
    ▼               ▼
┌────────┐   ┌──────────┐
│ SQLite │   │reportlab  │
│ 数据库 │   │ → PDF    │
│产品+价 │   │ → JPG    │
│格+客户 │   │          │
└────────┘   └──────────┘
                  │
                  ▼
            ┌──────────┐
            │ 微信送达  │
            │ PDF/图片  │
            └──────────┘
```

## 特点
- ✅ 纯 Python，无外部服务依赖
- ✅ PDF + 图片双输出
- ✅ 专业排版：深蓝色主题，表格隔行变色
- ✅ 自动计算折扣、合计
- ✅ 报价记录写入数据库可追溯
- ✅ 中文字体支持（文泉驿）
